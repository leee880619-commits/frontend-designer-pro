질문 몇 개에 답하면 바로 열리는 완성 폴더가 돌아옵니다. 백지도, 어려운 용어도 없습니다.

웹앱

사람들이 실제로 쓰는 화면

사람들이 클릭하고 입력하며 실제로 '쓰는' 화면. 버튼은 반응하고 폼은 처리되며, 로딩·빈 화면·오류까지 빠짐없이 챙깁니다.

여러 동료가 읽는 문서

C레벨·임원 보고서

결정을 내리는 분들께

짧게 보고 판단하는 자료. 화려함보다 '믿을 만한 인상'과 또렷한 핵심.

발표 덱 (HTML 슬라이드)

뒷자리에서도 읽히게

회의실 화면에 띄워 말로 설명하는 자료. 한 장에 메시지 하나, 뒷자리에서도 읽힙니다.

기타 (랜딩·마케팅 등)

위 넷에 안 맞으면?

한 번 더 여쭤, 가장 잘 맞는 방식을 찾아드립니다.

왜 "전문가급"인가요

화면 뒤에서 6명의 전문가가 순서대로 일합니다.

당신에게는 그림·쉬운 말·신호등 배지만 보입니다. 복잡한 장치는 무대 뒤에서만 돕니다.

01

아트디렉터

무드·위계와 '하지 말 것'을 정해 방향을 하나로 잡습니다.

02

디자인시스템 설계자

'한 곳만 바꾸면 전체가 바뀌는' 색·글꼴 구조를 만듭니다.

03

색채과학자

모든 색을 '잘 읽히도록' 계산해 뽑습니다 — 추측이 아니라 측정으로.

04

접근성·사용성 리드

키보드·포커스·스크린리더·터치 크기까지, 아무도 소외되지 않게.

05

데이터시각화 전문가

정직한 차트와 '결론을 말하는 제목'을 고릅니다.

06

프론트엔드 구현 엔지니어

빌드도 터미널도 없이 더블클릭으로 진짜 열리게 만듭니다.

내보내기 전, 자동 전수 점검 — 그리고 보정.

"잘 읽히는지 / AI가 바로 만든 티가 안 나는지 / 색맹 동료도 구분되는지"를 점검하고, 부족하면 알아서 고쳐 완성합니다. 당신이 숙제를 떠안지 않습니다.

  • 큰 글씨만 OK
  • 자동으로 고쳤음

설치

Claude Code 안에서 두 줄.

첫 줄은 장터를 등록하고, 둘째 줄은 플러그인을 설치합니다. 그게 전부입니다.

마켓플레이스 설치 (권장)

> /plugin marketplace add https://github.com/leee880619-commits/frontend-designer-pro

> /plugin install frontend-designer-pro

설치가 끝나면 /frontend-designer-pro가 자동완성 목록에 나타납니다.

사용법

질문부터 완성 폴더까지, 한 흐름.

한 화면 흐름으로 진행되고, 중간에 멈춰도 다음에 이어서 시작할 수 있습니다.

호출

/frontend-designer-pro 입력, 또는 "임원 보고서 만들어줘"처럼 자연어로. 첨부 파일(md·html)이 있으면 먼저 읽어 질문 수를 줄입니다.

질문에 답하기

딱 한 번, 꼭 필요한 것만 그림과 객관식으로. ★추천 기본값이 켜져 있어 "알아서 해주세요"를 골라도, 안 골라도 최고 품질이 나옵니다.

(선택) 샘플 비교

서로 다른 분위기 시안 3개를 나란히 비교하고 마음에 드는 하나를 고릅니다. 건너뛰어도 됩니다.

결과 폴더 저장

답한 위치에 사람이 알아볼 이름으로 저장됩니다. index.html을 더블클릭하면 브라우저에서 바로 열립니다. 터미널·서버·빌드 불필요.

결과물 폴더

사람이 알아볼 이름. 빌드 찌꺼기 없음.

dist/·.next/ 같은 빌드 찌꺼기 대신, 알아보고 고칠 수 있는 파일만.

내-결과물/
  index.html        ← 여기를 더블클릭하면 열립니다
  tokens.css        ← 색·글꼴이 한곳에
  assets/           ← 폰트·이미지
  samples/          ← (선택) 비교한 시안들
  편집-가이드.md     ← "이 숫자를 바꾸세요"
  검증-요약.md      ← 자동 점검 결과
  a11y-report.md    ← 잘 읽히는지 점검

터미널을 몰라도 됩니다

파일 탐색기에서 index.html을 더블클릭하면 열립니다.

한 곳만 바꾸면 전체가

색·글꼴을 바꾸려면 tokens.css 한 곳만. 함께 드리는 편집 가이드가 한국어로 안내합니다.

신호등으로 알려드립니다

🟢 잘 읽힘 · 🟡 큰 글씨만 OK · 🔴 자동으로 고쳤음 — 실생활 한 줄과 함께.

문제가 있으면 대신 고칩니다

색이 흐리거나 그림 설명이 빠지면, 완성 전에 자동으로 보정합니다.

자기완결

다른 것 안 깔아도, 오프라인에서도.

전문 지식을 모두 플러그인 안에 텍스트로 담아, 다른 스킬이나 네트워크에 품질이 좌우되지 않습니다.

혼자 동작

다른 디자인 스킬이 없어도 품질 그대로 — references/ 지식 번들이 자기완결입니다.

무빌드 더블클릭

외부 요청 0인 단일 index.html. file://에서도 더블클릭으로 열립니다.

한글 안전

사내망에서 CDN이 막혀도 한글이 깨지지 않도록 폰트 폴백을 갖춥니다.

말이 아니라 측정

대비·색맹·토큰을 더블클릭 점검 도구로 실측합니다 — 만들 때만 쓰고 결과물엔 넣지 않습니다.

정직한 보고

측정 안 한 것은 측정한 것처럼 말하지 않습니다. 교차브라우저·로딩속도는 실측했을 때만 단정합니다.

FAQ

자주 묻는 질문.

정말 코드를 몰라도 되나요?

네. 채팅창에 /frontend-designer-pro만 입력하면 됩니다. 결과는 index.html을 더블클릭하면 열립니다. 터미널은 필요 없습니다.

결과물을 나중에 수정할 수 있나요?

네. tokens.css 한 곳만 바꾸면 색·글꼴이 전체에 반영됩니다. 함께 드리는 편집 가이드가 어느 줄을 바꿀지 한국어로 안내합니다.

다른 디자인 플러그인을 먼저 깔아야 하나요?

아니요. 혼자서 완전히 동작합니다 — 필요한 지식이 안에 번들돼 있습니다.

무엇을 만들 수 있나요?

웹앱·분석 보고서·임원 보고서·발표 덱, 그 외 랜딩·마케팅 페이지도 만듭니다.
