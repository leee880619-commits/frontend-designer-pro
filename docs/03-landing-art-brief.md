# 03 — 랜딩 페이지 Art Brief (SSOT)

> 대상 산출물: `frontend-designer-pro`의 **GitHub 공개 설명서 페이지**(GitHub Pages, gh-pages 브랜치, 한 페이지 랜딩, 한국어/영어 토글).
> 이 한 장이 색·글꼴·레이아웃·모션의 단일 상류 입력이다. 작성 2026-06-02.

## 1. 도메인 / 의도
- 도메인: **기타 §4.5 → 프로젝트 랜딩/설명서**(마케팅·설득 톤 일부 허용, 단 가독성 하한선 위에서만).
- 메타 제약: **디자인 도구의 얼굴** → 페이지 자체가 의도적으로 다듬은 디자인 수준을 증명해야 한다(슬롭이면 제품 신뢰 직격).
- 청중: ①Claude Code를 쓰는 개발자 ②플러그인 결과를 받는 비개발자(사내 한국어 조직, KRAFTON). 둘 다 친근하게, 설명은 명확하게.

## 2. 미적 방향 — soft-modern × editorial 명료성 하이브리드
- 무드 키워드 3개: **친근한 · 또렷한 · 신뢰감 있는**.
- 연상 레퍼런스: 잘 만든 개발자 제품 랜딩(둥근 카드·은은한 그림자) + 잡지식 강한 위계(큰 헤드라인 → 데크 → 본문).
- 폰트: 한글 **Pretendard 1순위 + system-ui 폴백**(CDN 단독 의존 금지). 의도성은 **모노 아이브로 라벨(letter-spaced uppercase)** + 극적인 크기/굵기 대비 + 비대칭 리듬으로 표현(기본폰트 방치 아님). 숫자·명령어는 `ui-monospace`.
- 하지 말 것 3개: ①보라→파랑 그라데이션 on 화이트(A1) ②균등 3카드 나열(A3) — 대신 featured+리스트 비대칭 ③의미 없는 이모지 아이콘 장식(A4) — 신호등 🟢🟡🔴만 의미 신호로 허용.

## 3. 색 (장식면 무드 / 본문은 게이트 통과값 — 측정 완료)
- 라이트: ink `oklch(0.24 0.02 264)`#1a1f29 · paper `#f9fafd` · band `#f1f3f8` · 액센트 brand `oklch(0.58 0.15 264)`#4c75d2(라인/UI), button `#325cbd`(흰 글자), 인라인 링크 `#21459d`.
- 다크(`prefers-color-scheme` 스왑): paper `#13161c` · ink `#e4e8ef` · 링크 `#bdd8ff` · 버튼은 라이트와 동일(흰 글자 on `#325cbd`).
- 따뜻한 보조 액센트 coral `#d45b3d` — **장식 전용(아이브로 바·점), 텍스트 절대 안 올림**. → 60-30-10: 주 액센트 brand 1 + 장식 warm 1 = 2개 이내.
- 측정 결과: 본문 Lc 100/15.8:1, 버튼 라벨 Lc 86/6.2:1, 인라인 링크 Lc 89/8.7:1 등 전 쌍 APCA AND WCAG 통과(`docs/`의 측정 로그 기준).

## 4. 레이아웃 / zone 경계
- expression zone(장식 자유): 히어로 배경 질감, 아이브로 바, 섹션 구분 장식.
- data zone(절제): 본문 설명·설치 명령·폴더 트리·FAQ — 명료·고대비.
- 비대칭 다단: 히어로 좌측 정렬 큰 헤드라인 + 우측 설치 카드. 섹션은 featured 블록 + 보조 리스트.

## 5. 모션 예산
- 진입 1회 페이드/슬라이드 ≤200ms, hover transform ≤200ms. 무한 애니메이션·`transition:all` 금지.
- `@media (prefers-reduced-motion: reduce)`에서 모션 0(스크롤 진입 효과 포함 전부 무효).

## 6. 의도성 증거(왜 이 선택인가) — anti-slop A-meta 대응
- 폰트: Pretendard+모노 아이브로 → 한영 혼용 가독 + 개발자 도구 톤을 CDN 없이 오프라인 안전하게.
- 색: soft 블루 단일 액센트 → 친근함과 신뢰를 동시에, 60-30-10로 강조를 강조답게.
- 레이아웃: 비대칭 featured → 균등 나열(슬롭)을 피하고 "무엇이 먼저인지"를 위계로 전달.

## 7. 기술 제약(무빌드)
- 단일 `index.html`(인라인 CSS/JS, 외부 요청 0). gh-pages 루트에서 서빙, 더블클릭 file://로도 열림.
- 한국어 `word-break:keep-all`·`line-height ≥1.6`. 영어 토글 시 `word-break:normal`로 스코프.
- 영어/한국어 토글은 progressive enhancement(JS 없으면 한국어 기본 표시). a11y 1급 7키 포함.
